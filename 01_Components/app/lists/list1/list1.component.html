Маршрутизация

Маршрутизация - отслеживание измненений URL в адрасной строке и перенаправление пользователя на определенное представление(template)

4 шага по настройке маршрутизации

  1. < head >
    <base href="/" >
  2. import {RouterModule} from "@angular/router" 
  3. const routes: Routes = [ 
    { path: "list1", component: List1Component }, 
    { path: "list2", component: List2Component },
    { path: "", redirectTo: "list1", pathMatch: "full" }
 ]

 @NgModule({
    imports: [
        BrowserModule,
        RouterModule.forRoot(routes)
    ]
  })
  4. <nav>
	<a routerLink="/list1" routerLinkActive="active">Lesson 1</a>
	<a routerLink="/list2" routerLinkActive="active">Lesson 2</a>
 </nav>

Описание

  • Пункт 4 меняет адресную строку при этом не перенаправляет нас на новую страницу
  • пункт 3 подхватывает изменения в адресной строке и перенаправляет нас на нужный компонент, темплейт которой вставляеться после <router-outlet ></router-outlet>
