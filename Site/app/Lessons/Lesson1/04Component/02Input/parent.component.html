Особенности примера

Как видим для использования дочернего компонента в родительском достаточно в родительском написать имя selector дочернего компонета. Для передачи данных в дочерние компоненты есть несколько способов:

  1. Передача строки или числа [name]="'First counter'", [counterValue]="5". В правой[name] части мы используем имя переменной(свойства) дочернего компонента которой мы хотим присвоить значение. В правой части "'First counter'" мы передаем переменную(свойство) родительского обьекта выделенное "" либо ''. В нашем случае мы передаем "'First counter'", "5" это строка и число.
  2. Так же мы можем передвавать переменную(свойство) родительского обьекта выделенное "" либо ''. Если использовать имя несуществующей переменной то мы получим ошибку.
  3. Перейдем к дочернему компоненту. Тут используться несколько видов привязки. Name = {{name}} в данном случае используеться одностороння привязка и вместо {{name}} будет подставляться значение свойства name дочернего компонента. Как мы видим у дочернего компонента есть свойство name name: string = "default child component name " которое типизировно типом string и имеет дефолтное значение "default child component name ". Так же необходимо обратить внимание на метаданные дочернего компонента inputs: ["name"].
  4. Так же еще один способ указать что значение свойства будет задаваться из вне это декорировать его @Input() @Input() counterValue: number = 0;
  5. есть возможность изменить смапить значение передаваемого значения и принимаемого свойства
  6. Соблюдение всех этих условий дают нам возможность пердавать данные из родительского компонента в дочерний и использовать их в дочернем компоненте.
  7. Так же мы видим что в дочернем компонете используеться привязка события на кнопке (click)="increment()" где на событие (click) привязываеться функция "increment()" из класа дочернего компонента
